Important FAQ:

What is document object model(DOM)?

The document object model represents the HTML document as a tree of objects which are called nodes. Using the dom developer can make dynamic websites, handle events and make real-time updates to the website.

How do You select an element from the DOM?

To Select an element from dom we can use several methods: For instance,

  1. document.getElementById()
  2. document.getElementsByClassName()
  3. document.getElementByTagName()
  4. document.querySelector()
  5. document.querySelectorAll()
What is event delegation in the context of DOM and why is useful?

Event Delegation is a technique in JavaScript where a parent element is used to manage event listeners for its child elements. Instead of attaching event listeners to individual child elements, you add a single listener to a common parent element. The parent listens for events from its children using event bubbling.

How do you manipulate an element's attributes and styles using DOM?

Manipulating Attributes:

  1. getAttribute()
  2. setAttribute()
  3. removeAttribute()

Manipulating Styles:

  1. classList.add()
  2. classList.remove()
  3. element.style.style_name = 'value of the style'
